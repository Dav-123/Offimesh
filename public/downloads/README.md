# APK Downloads Folder

## How to add the APK

1. Export your Android APK file from Android Studio or your build system
2. Rename it to: `offimesh.apk`
3. Place it in this folder: `public/downloads/offimesh.apk`
4. Commit and push to GitHub

The download button on the website will automatically serve it from:
`/downloads/offimesh.apk`

## Filename expected
`offimesh.apk`
